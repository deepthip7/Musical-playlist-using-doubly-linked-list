# Musical-playlist-using-doubly-linked-list

Project Description
This project implements a Musical Playlist Management System using a Doubly Linked List data structure. It allows users to add, remove, play next, and play previous songs efficiently. The project demonstrates the practical use of data structures in real-world applications like music players.

->Objectives
To understand and implement Doubly Linked List
To perform dynamic insertion and deletion of songs
To enable forward and backward traversal of the playlist
To build a simple music playlist simulation

->Technologies Used
Programming Language: Java(Backend),html, css, javasricpt(frontend)
IDE: IntelliJ IDEA
Data Structure: Doubly Linked List

-> Features
Add a new song to the playlist
Remove a song from the playlist
Play the next song
Play the previous song
. Display the entire playlist
